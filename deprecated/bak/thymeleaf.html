SpringMVC 配置文件下配置 Thymeleaf

  • 依赖 : thymeleaf-spring5, spring-context-support

表达式语法

  • ${} : 变量表达式
  • *{} : 选择表达式
  • @{} : 超链接url表达式
  • #{} : 消息表达式

常用标签

  • th:text

    : 用于文本的显示

  • th:utext

    : 和 th:text 类似但可以解析 HTML 文本

  • th:if

    : 用于判断条件, 可以与 and, or, !, not 一同使用, 如果条件为 true 则显示

  • th:unless

    : 与 th:if 作用相反, 如果条件为 false 则显示

  • th:switch th

    :case

  • th:each : foreach

  • th:selected

    : 选中 select 的选项

  • th:value

    : 属性赋值, 用于 option 和 input

  • th:action

    : 定义后台控制器的路径

  • th:src

    : 外部资源引入

  • th:href

    : 定义超链接
  • th:remove : 用于删除
    • all : 删除所在标签及其全部内容
    • body : 不删除标签, 但删除子标签和其内容
    • tag : 删除所在标签, 不删除子标签
    • all-but-first : 删除除了第一个子标签以外的其他子标签
    • none : 什么都不删
  • th:object : 对象绑定
