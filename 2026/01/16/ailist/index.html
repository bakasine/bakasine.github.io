AI 工具列表

2026-01-16

vibe cli

Chat

  • Gemini 直接根据谷歌账号登录, 不需要额外登录不容易过期登出, 没有CF验证
  • Copilot 无需登录, 但是有点智障
  • ChatGPT 登录麻烦, 有效期巨短, 注册麻烦
  • Claude 注册麻烦

Vibe Coding

  • Antigravity 谷歌白嫖, 多个模型免费使用
  • OpenCode 开源免费, 可以装一些插件使用其他家的付费模型
  • Cursor 试用期短, 前端很强, 太贵
  • Claude 据说写代码很强, 但是没用过
  • Vscode 原生vscode, 有点免费额度但是完全不够用
  • Trae 字节, 据说免费
  • Codebuddy 腾讯

Cli

  • GeminiCli 谷歌, 有一点点免费额度
  • ClaudeCode 要钱
  • OpenCode 开源免费
  • Codex openai, 要钱
  • Codebuddy 腾讯
  • Aider 开源
  • QwenCode 阿里
  • Copilot 没用过
