用法

miningrigrentals 仅支持BTC LTC DOGE ETH BCH等支付方式，所以我们就选择手续费最低的DOGE作为矿机租赁费用来进行支付

充值: Balance -> Deposit Addresses

矿池参数: Favorite Pools -> Add A Pool

参数名 简介
Name 备注
Type 算法
Pool Host:Port 矿池地址:端口
Workername 钱包地址

租赁算力: Favorite Pools -> 左侧Marketplace -> 选择对应算法
